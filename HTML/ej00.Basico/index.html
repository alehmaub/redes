Cuerpo principal del contenido

La etiqueta main es en HTML5 una etiqueta semantica. Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella. Define el contenido principal de la página. Unicamente puede haber una sola etiqueta como esta en todo el documento. El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de página, los menues de navegación y los pie de página). Es usado por buscadores para indexar. Conviene usar aquí textos que ayuden a encontrar la página luego de procesos de indexación de los buscadores.

Sección etiquetas semánticas estructurales más importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las etiquetas semánticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semánticas:

  • Los distintos tipos de contenido se pueden asociar a la etiquetas SECTION
  • Las uidades de contenido se asocian a las etiquetas ARTICLE O sea que un SECTION anida mlitiples ARTICLES
  • Las etiquetas HEADER Y FOOTER pueden se anidadas dentro d eotras HEADER o FOOTER Osea un FOOTER podría anidar otro HEADER y FOOTER
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegación del documento
  • Reservar etiqueta h1 oara usar dentro de etiquetas top level como main, header o footer.
